<h3 align="center">Description</h3>

<p align="center">
Simple application for generation anime schedule created on Blazor Server (.NET 6) and created for my own group in VKontakte - <a href="https://vk.com/anilife">AniLife</a>.</br>
Application using API of <a href="https://anilist.co/">Anilist</a> for get all airing anime schedule in selected day and API of <a href="https://shikimori.one/">Shikimori</a> for fastest way to get russian title.
</p>
